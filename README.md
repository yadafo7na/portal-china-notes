# portal-china-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存放，不依赖后端服务，可直接通过浏览器打开或部署至任意静态托管平台。

## 项目简介

portal-china-notes 是一个纯前端 HTML 页面集合。仓库中的每个页面均为独立文件，内容涵盖信息整理、工具页面、资料索引等。项目旨在以最简方式存储和展示这些页面，方便日常使用、分享与版本管理。

## 目录说明

```
.
├── index.html          # 仓库首页或入口页面
├── pages/              # 存放主要 HTML 页面文件
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：核心页面目录，每个文件为一个独立页面。
- `assets/`：存放页面引用的公共资源，按类型分类。

## 页面归档说明

- 所有页面均为静态 HTML，无外部依赖（除公共资源外）。
- 页面之间相互独立，不共享状态或数据。
- 部分页面可能引用 assets 目录下的资源文件，请确保路径正确。
- 页面内容以中文为主，部分页面可能包含英文或其它语言资料。

## 使用方法

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/portal-china-notes.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件。
3. 也可将整个仓库部署至 GitHub Pages 或其他静态托管服务。

## 维护说明

- 添加新页面时，在 `pages/` 目录下新建 `.html` 文件，并确保引用资源路径正确。
- 如需修改公共资源，请同步更新所有引用该资源的页面。
- 建议保持页面结构简洁，避免过多内联样式或脚本，以便后续维护。
- 提交前请检查页面是否能在本地正常打开。
